1. Carregando imagem Local
            * Crie um arquivo HTML da pasta <ex003>
            * Busque uma imagem .png do símbolo HTML (30kb)
            * Selecione uma imagem sem direitos autorais
            * No HTML use a tag <img src="caminho-local" alt="texto alternativo">
                (ctrl + espaço)
            * Use o GIMP para 200 de largura nas imagens e exporte (14kb)
            * Faça o mesmo com a imagem da CSS

2. Carregando imagem externa:
            * Baixe uma logo de javascript copiando o endereço da imagem
            * no <img src="cole-o-caminho-url" alt="texto alternativo">
            * Risco do servidor cair e seu site ficar sem imagem*
